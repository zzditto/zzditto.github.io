hexo使用教程

发表于 2019-07-30 | 更新于: 2019-09-23 | 分类于 Learn |

windows环境

安装node.js

https://nodejs.org/en/

安装git

https://git-scm.com/download/win

使用管理员权限操作命令行

开始菜单点击鼠标右键，选择powershell(管理员)

执行命令安装hexo

初始化hexo。路径为自定义路径【这里是在D盘下新建blog文件夹】

cd到博客目录，执行安装程序

安装完成，打开web页面

hexo s 或 hexo server

浏览器访问 localhost:4000

将hexo部署到github

点击 git Bash 【需要上传的命令都需要git bash，使用cmd.exe会报错】
点击github页面，新建仓库。仓库名为 用户名.github.io。然后复制仓库总网址

在blog目录下安装git插件

编辑blog的_config.yml文件【设置type为git，添加repo和branch】

在git bash中设置github的邮箱和用户名

如果不输入，直接hexo d上传，报错

在命令行输入：

上传命令：

hexo d

然后会提示输入github的用户名和密码，登录，稍等片刻，上传成功

使用github访问

https://zzditto.github.io/

创建第一篇博客

在blog即博客安装的总目录下，鼠标右键，打开git bash窗口

创建md文件

在md文件中添加内容

打开win下的博客目录，使用makedown编辑器打开此文件，添加内容，ctrl+s保存

清除原来缓存

重新生成html页面文件

启动

同步至github

访问

本机：
localhost:4000

网络:
http://zzditto.github.io

绑定域名后:
http://www.zzditto.cn/
