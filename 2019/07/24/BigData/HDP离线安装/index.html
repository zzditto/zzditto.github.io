HDP离线安装

发表于 2019-07-24 | 更新于: 2019-08-29 | 分类于 大数据 ， HDP |

注意
在实际生产环境下，需要离线安装，但是一般都是自定义yum源，将所需的rpm软件包及其依赖提前准备好并制作一个yum源服务器，供集群的各节点使用。本文采用的方法很笨，具体yum源的制作请看http://www.zzditto.cn/2019/08/19/HDP%E7%A6%BB%E7%BA%BF%E5%AE%89%E8%A3%85%E5%88%B6%E4%BD%9C%E6%9C%AC%E5%9C%B0yum%E6%BA%90/,下文对软件的安装在yum源制作完成之后，都可使用yum -y install packagename 安装。

1.检查系统版本

HDP2.6需要centos7版本
注：系统需要linux的虚拟化主机版或者桌面版。最小化安装缺乏mysql安装的部分依赖包

cat /etc/redhat-release 适用于Redhat系linux

2.检查分区硬盘大小

系统要求：【需在装载系统时分好】
root：100G
swap：同内存大小
home：40G
数据盘：剩余容量
