ftp数据导入整体流程

  1. ftp数据下载，下载的为sql文件
  2. 将存于本地的sql文件上传至hdfs
  3. 执行本地的sql文件，将其导入mysql中
  4. sqoop将mysql的数据导入hdfs
  5. 清除mysql表数据
