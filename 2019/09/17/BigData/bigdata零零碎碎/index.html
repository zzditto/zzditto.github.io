个人电脑虚拟机作为测试集群节点

桥接模式是将主机当成为一个交换机，桥接模式的虚拟机和主机都可以看成通过这个交换机连接到外部网络，这两个机器不会互相影响，完全独立。但是得设置虚拟机的ip和网关和主机的ip和网关在同一网段

linux安装anaconda之后，主机名之前会出现base

原因：安装conda之后，每次启动终端，都会启动conda的base环境。conda的环境可以使用conda env list查看。关掉conda环境的自动启动即可。

解决：
(base) [root@vm50 ~]# conda config --set auto_activate_base False

如果后续需要跑py程序，设置base环境自动启动即可
